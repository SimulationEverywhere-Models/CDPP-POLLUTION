

La carpeta contiene 8 ejemplos de simulacion. 
Desde e1 hasta e8.

Cada Simulación consta de 5 archivos. 
	ex.drw 	= Salida del drawlog.exe
	ex.ent 	= Archivo de mensajes externos
	ex.ma	= Modelo
	ex.val  = Valores Iniciales
	simux.bat = Script que corre el simu.exe y drawlog

El archivo e1.ma contiene el modelo original
